Was zählt wirklich: Gesamtgewinn über 25 Jahre

Weshalb die Amortisationszeit alleine ein unvollständiges Bild ergibt: Mit ZEV investieren Sie mehr (Zähler, Server, Software) — deshalb dauert die Amortisation etwas länger. Wohin fliesst der Solarstrom?

Funktionsweise
Die Solaranlage produziert Strom. Dieser wird auf 3 Wege aufgeteilt: Das Haus verbraucht einen Teil selbst, ein Teil wird direkt an die Mieterinnen und Mieter verkauft (ZEV), und der Rest wird ins Stromnetz eingespeist. Die Prozentsätze können Sie links im Regler anpassen — der Rechner zeigt Ihnen sofort, wie sich dies auf die Einnahmen auswirkt.

ZEV Kosten

Weshalb verursacht ZEV Mehrkosten?
Damit Strom direkt an die Mieterinnen und Mieter verkauft werden kann, benötigt jede Wohneinheit einen eigenen intelligenten Zähler, einen Server zur Messdatenerfassung sowie eine Software für die Abrechnung. Dies ergibt einmalige Installationskosten sowie jährliche Betriebskosten.
